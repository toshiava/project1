# wareki.py

西暦を和暦に変換するPythonスクリプトです。

## 使い方

```bash
python wareki.py <西暦年>
```

## 実行例

```bash
python wareki.py 2026
# 2026年 → 令和8年

python wareki.py 1989
# 1989年 → 平成元年

python wareki.py 1926
# 1926年 → 昭和元年
```

## 対応元号

| 元号 | 開始年 |
|------|--------|
| 令和 | 2019年〜 |
| 平成 | 1989年〜 |
| 昭和 | 1926年〜 |
| 大正 | 1912年〜 |
| 明治 | 1868年〜 |

1868年（明治元年）以前の年は対応していません。

## 要件

- Python 3.7以上

## エラーケース

| 状況 | メッセージ例 |
|------|-------------|
| 引数が整数でない | `エラー: 'abc' は整数ではありません。` |
| 対応範囲外の年 | `エラー: 1800年は対応範囲外です（1868年以降を指定してください）。` |
| 引数の数が不正 | `使い方: python wareki.py <西暦年>` |
